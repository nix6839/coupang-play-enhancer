# 쿠팡플레이 개선 확장

[쿠팡플레이 웹사이트](https://www.coupangplay.com/home)의 기능들을 개선해주는
브라우저 확장 프로그램입니다.

## 기능

- 방향키(<kbd>←</kbd>, <kbd>→</kbd>) 비디오 시간 조정 단위를 10초에서 5초 단위로 수정
- 재생속도 전환 단축키(<kbd><</kbd>, <kbd>></kbd>) 추가
